Definicion de hojas de estilo

El código HDL sufre entonces una revisión de código o de auditoría. En la preparación de síntesis, la descripción HDL es objeto de un conjunto de fichas automatizadas. Los inspectores informaron las desviaciones de las directrices de código estandarizadas identificar posibles construcciones de código ambiguos antes de que puedan causar una mala interpretación y comprobar si hay errores de codificación lógico común, como colgando puertos o salidas en cortocircuito. El proceso ayuda a resolver los errores antes de que se sintetiza el código.
